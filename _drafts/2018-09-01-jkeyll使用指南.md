## 参考文档
主要参考[官方文档](https://jekyllrb.com/docs/structure/)就行了。

## 本地编译
在主文件夹中，
```
jekyll server
```
若端口被占用，则自己设定端口：
```
jekyll server --watch --port 4400
```
