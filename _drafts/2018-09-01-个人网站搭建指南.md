## 参考文档
主要参考[官方文档](https://jekyllrb.com/docs/structure/)就行了。

## 本地编译
在主文件夹中，
```
jekyll server
```
若端口被占用，则自己设定端口：
```
jekyll server --watch --port 4400
```

## Documentation
### image
```
<img src='/img/deep_learning_course/merg_sort.png' width='600' >
```
1. width, src 需要包含在''里面
2. 空格分隔各个量，不能是common
3. 地址前要叫“/”
4. 只加width就行，不需要加height
