# Get start
尝试[官网指南](https://pro.ant.design/docs/getting-started-cn#%E5%AE%89%E8%A3%85)不太行，改用下面步骤
```
# 1. 安装npm， 会在Node.js 安装过程中自动装好
# 查看版本
node -v
npm -v
# 2. 安装create-umi
npm install create-umi -g
# 3. 运行create-umi
create-umi 
# 之后步骤按照官网
```
